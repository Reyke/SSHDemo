Q1. How to pop up a alert message ?

A1. call JS method : popAlert,
Demo :
popAlert("This is my title", "This is my message");

Q2. How to pop up a confirm message ?

A2. call JS method : popConfirm,
Demo :
popConfirm("This is my title", "This is my message",
    function(){
        // This method will be called while OK clicked.
    },
    function(){
        // This method will be called while Cancel clicked.
    });

Q3. How to navigate to other page ?

A3. call JS method : navigate,
Demo :
navigate(THIS_IS_THE_URL);

Q4. How to edit internationalization file ?

A3. install JInto in your eclipse, you can install this plugin from this URL : http://www.guh-software.de/eclipse/

Q5. How to use internationalization property in HTML code ?

A3. you can create record in the i18n/applicationResources_*.properties, and then set the value or innerText of your component to #{ApplicationResource.XXXXXXXXXX}, our framework will fill this value with the correct type of language.
Demo : 
login.button.login=Login		#this is the record in the applicationResources_en_US.properties

login.button.login=登陆		#this is the record in the applicationResources_zh_CN.properties
<input type="button" class="QRTbutton" value="#{ApplicationResource.login.button.login}" onclick="dologin()" />

Q6. How to create a button with JQuery-UI style ?

A3. you can add class QRTbutton to the button you created.
Demo :
<input type="button" class="QRTbutton" />

Q7. How to execute a function while loading a screen ?

A3. you can define a function inited() in your JS code, this method will be called while loading your screen.
Demo :
function inited() {
// codes here will be called while loading this screen.
}

Q8. How to execute a function while unloading a screen ?

A3. you can define a function destroy() in your JS code, this method will be called while unloading your screen.
Demo :
function destroy() {
// codes here will be called while unloading this screen.
}
